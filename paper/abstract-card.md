# Cassandra abstract card

Working title: Cassandra: From Governance Infrastructure to Evidence Infrastructure

## Short abstract

Cassandra is a research-only observatory for public European list-of-lists and national trusted-list artifacts. It asks how public legal-technical governance state can be made observable as evidence infrastructure without converting observation into legal supervision. The system records dated public-source collection telemetry, normalizes parseable XML for structural comparison, emits cautious aggregate diffs, packages run evidence, verifies EATF/AEP receipts for package integrity, and publishes a static dashboard index with explicit claim boundaries. The current public series contains four dated runs through 2026-05-27, all with EATF receipt status `ok`, and fixture-backed tests for stable/no-change behavior, normalized-hash changes, provider and service inventory movements under hashed identifiers, fetch failures, non-XML inputs, missing signing inputs, dashboard states, and forbidden overclaiming wording. The paper argues for a methodological distinction: public governance artifacts can be observed, hashed, bundled, signed, timestamped, displayed, and independently checked while preserving the difference between evidence integrity and legal interpretation.

## One-sentence contribution

Cassandra demonstrates how machine-readable public governance artifacts can be transformed into reproducible evidence packages and dashboard telemetry without asserting legal status, signature validity, supervisory approval, compliance judgment, public alerting, or absence of legally relevant change.
